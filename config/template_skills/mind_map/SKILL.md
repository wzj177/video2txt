# 思维导图模板

## 使用场景
当需要把长篇音视频转录整理为结构化的思维导图时使用。模板强调关键信息提炼、节点数量约束以及清晰的 Markdown 列表格式。

## system_prompt
# 任务
基于转录内容生成思维导图，核心要求：
- 结构清晰，层级合理
- 聚焦核心概念与关键要点

# 约束条件
1. **语言**：简体中文
2. **结构层次**：
   - 主节点：3-5个核心主题
   - 子节点：每个主节点下2-4个要点
   - 总节点数：控制在9-15个
3. **节点要求**：
   - 使用简洁短语，避免完整句子
   - 体现{domain}领域特色
   - 突出核心观点和关键概念
4. **时间点**：
   - 主要节点和关键子节点需要附带时间点
   - 时间点格式为 `mm:ss`，放在节点文本后，例如：`- 相机事件 \`00:12\``
   - 时间点必须来自转录中的时间范围
5. **事实约束**：
   - 只允许使用转录中出现的信息

# 输出示例
# 思维导图：{{主题标题}}

- {{主节点1}} `00:12`
  - {{子节点1.1}} `00:45`
  - {{子节点1.2}} `01:20`
- {{主节点2}} `02:10`
  - {{子节点2.1}} `02:45`
  - {{子节点2.2}} `03:10`
- {{主节点3}} `04:05`
  - {{子节点3.1}} `04:40`
  - {{子节点3.2}} `05:15`

请严格按照模板输出，不要添加额外解释。

## user_prompt
请为以下{domain}内容生成专业的思维导图：

### 转录内容（含时间点）：
{timed_transcript}

### 分析要点：
- 主要领域：{domain}
- 内容风格：{content_style}
- 目标受众：{target_audience}
- 核心话题：{key_topics}

请生成结构清晰、层次分明的思维导图。
