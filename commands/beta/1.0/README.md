# 🎬 AI视频文本分析

## 📋 项目简介

这是一个基于AI的视频内容分析和转换工具，能够将视频转换为多种格式的文本内容，包括字幕、摘要、思维导图等。


## 🚀 核心功能

### 1️⃣ 基础功能
- **视频转音频**：支持多种视频格式，自动缓存音频文件
- **音频转文字**：使用Whisper模型，支持中英文，智能字幕生成
- **关键帧提取**：智能选择重要时刻的画面，30秒最小间隔避免重复
- **字幕生成**：生成高质量SRT格式字幕文件

### 2️⃣ 内容生成（优化版）
- **小红书笔记**：专业化内容结构，信息密度高，支持配图
- **思维导图**：多层结构，逻辑清晰，Markmap兼容格式
- **关键时刻**：精准识别，时间精确到秒，3-8个关键片段
- **内容结构分析**：深度解析视频组织方式和叙事框架
- **可信度评估**：5维度专业评估（数据来源、逻辑性、客观性等）
- **价值评分**：6维度量化评估内容价值

### 3️⃣ 平台适配（优化功能）
- **单平台适配**：针对抖音/小红书/B站的专门优化
- **全平台一键生成**：同时生成三个平台的适配内容
- **平台特色**：针对各平台用户习惯和内容特点深度优化
- **格式标准**：符合各平台的内容规范和推荐机制

### 基本使用


### 参数说明
```bash
# 必需参数
-i, --input              # 输入视频文件路径或URL

# 可选参数
--output_dir            # 输出目录
--verbose               # 启用详细日志输出
--whisper_model         # Whisper模型 (tiny/base/small/medium/large)
--gpt_model            # GPT模型名称
--language             # 音频语言代码 (zh/en)
--api_key              # API密钥
--api_base             # API基础URL

# 输出格式
--format               # 输出格式 (markdown/markmap/both)

# 扩展功能
--key_moments          # 生成关键时刻标记
--credibility          # 生成可信度分析
--structure            # 生成内容结构分析
--value_rating         # 生成内容价值评分

```

## 📁 文件结构

```

├── video2txt.py              # 
├── requirements.txt          # 依赖包列表
```
